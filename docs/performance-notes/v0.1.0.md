# RustDroid Performance Notes: v0.1.0

Reference cold host-fixture run from local verification on April 2, 2026:

- boot: about 65.5s
- install: about 12.4s
- launch: about 8.4s
- total run: about 86.3s

CI guardrails use conservative maximums rather than exact-match targets:

- boot <= 90s
- install <= 20s
- launch <= 15s
- total <= 120s

These thresholds are meant to catch major regressions on generic Linux runners, not micro-optimizations on a tuned local machine.
